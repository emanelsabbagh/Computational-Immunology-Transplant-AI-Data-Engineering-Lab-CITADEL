# scRNA-seq Notebooks

This folder contains notebooks for single-cell RNA sequencing analysis, including:

- Data loading
- Quality control (QC)
- Normalization
- PCA and UMAP
- Clustering
- Cell type annotation (T, B, NK, monocytes, dendritic)
- Gene markers and differential expression

These notebooks support the Immune Trajectory Atlas project and future single-cell transplant immunology analyses.

All data are public and contain no PHI.
